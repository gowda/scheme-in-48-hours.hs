# scheme-in-48-hours.hs

Following instructions from [Write Yourself a Scheme in 48 hours](https://upload.wikimedia.org/wikipedia/commons/a/aa/Write_Yourself_a_Scheme_in_48_Hours.pdf)

## Setup
```bash
$ stack setup
```

## Run test
```bash
$ stack test
```

## Run
```bash
$ stack run "hello world"
```
